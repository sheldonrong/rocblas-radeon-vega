# rocblas build for Radeon Vega based GPUS

Supported GPUs

* Radeon Vega 64/56
* Radeon Pro Vega 64
* Radeon VII
* Padeon Pro Vega II
* Instinct MI25
* Instinct MI50/60

Including rocblas build on version

* rocm 6.4
* rocm 7.2

Copy the TensileLibray files to your current rocm installation folder to enable these GPUs
